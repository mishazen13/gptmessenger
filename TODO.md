# TODO List - Исправление загрузки файлов через локальную сеть

## Задачи:
- [ ] 1. Изменить server/index.js - возвращать относительные пути для файлов
- [ ] 2. Изменить src/lib/api.ts - добавлять базовый URL к путям файлов
- [ ] 3. Протестировать

## Описание проблемы:
При доступе к приложению с другого ПК в локальной сети, файлы (аватарки, картинки, вложения в чате) не загружаются, потому что сервер возвращает URL с `localhost` или неправильным хостом.

## Решение:
1. Сервер возвращает относительные пути (`/uploads/filename.ext`)
2. Клиент добавляет базовый API URL к относительным путям при отображении файлов
